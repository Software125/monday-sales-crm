## Overview

**Monday CRM** is a customizable CRM solution built for teams that want a visual workspace for sales pipelines, customer tracking, workflows, tasks, and collaboration.

It is useful for teams that want CRM features combined with project and workflow management.

---

## Key Features

- Sales pipeline management
- Lead and account tracking
- Task management
- Team collaboration
- Workflow automation
- Custom dashboards
- Customer communication tracking
- Reporting tools
